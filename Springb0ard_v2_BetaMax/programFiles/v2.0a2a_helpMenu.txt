Springb0ard Help Menu:
-----------------------------------------------------------
  Command   |   --Description                             |
-----------------------------------------------------------
bb (or) btop    --Graphical display of the server's data statistics

sb-help         --Displays this help menu

sb-update       --Updates the repository from github using git

sb-log          --Displays the Springb0ard system log

db-latest       --Displays the latest features from the last Springb0ard update

sb-recovery     --Brings Springb0ard back from the dead in the event of update failure

install-scp3r   --Prepares the Secure Copy/SCP utility by creating ssh keys if they do not already exist

scp3r           --SCP export utility, long version sets up stored variables that can be accessed by qscp 

qscp            --Quick SCP Export utility, just add the file name to export and press enter.

hi              --Runs Debian's security and software updates.

bootmail        --Reboots Postfix and Dovecot

maillog         --Displays the last 100 lines of the postfix maillog

sb-deploy       --This is used ONLY in the root email server installer after root is configured.

sb-test         --Run test script

sb-etest        --Edit test script
